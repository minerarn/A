About This Blog System

Project Overview

This blog system is a comprehensive web application built as part of a semester-end assignment. It demonstrates the integration of modern web technologies including Spring Boot, MyBatis, Thymeleaf, and Spring Security.

Features
  • Admin Panel: Full CRUD operations for posts and categories
  • User Interface: Clean, responsive frontend for reading posts
  • Security: Role-based access control with Spring Security
  • Search: Multi-criteria search for posts
  • Statistics: View counts and category post counts
  • Logging: AOP-based logging for system operations
Technologies Used
  • Spring Boot 3.x
  • MyBatis for database access
  • Thymeleaf for server-side templating
  • Bootstrap 5 for responsive UI
  • Spring Security for authentication and authorization
  • Aspect-Oriented Programming for logging
  • MySQL database
